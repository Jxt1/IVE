# IVE
IVE: Accelerating Subgraph Matching with Isolated Vertices Exploration

# Execute
Execute the following command to run the program.
```bash
export LD_LIBRARY_PATH=.:$LD_LIBRARY_PATH
./IVE -d ./data/tiny.graph -q ./data/tiny_query.graph -num 100000
```
# Output
The output includes the number of the non-isolated vertices, matching order, the number of matched subgraphs and the search space size.
